## Visión Conceptual
RizoRhythm es un marco para un sistema de diagnóstico agrícola que correlaciona **ritmos mecánicos de las plantas** con variables de suelo (humedad, nitrógeno) para detectar estrés de forma proactiva y adaptativa.

## Arquitectura en Tres Capas
1.  **Sensado de Suelo:** Humedad y nitrógeno en la rizósfera.
2.  **Sensado de Ritmo:** Movimientos fitomecánicos (ej: con acelerómetro).
3.  **IA Contextual en el Edge:** Diagnóstico y sugerencia de acción (ej: riego).
